# Shader Window Website

背景動画の上に複数のウィンドウを配置し、ウィンドウ部分だけ動画が見えるインタラクティブなWebサイト。  
WebGLシェーダーを使ったエフェクトをウィンドウに適用することを目的としている。

## 使用技術

- HTML / CSS
- JavaScript
- Canvas API
- WebGL / GLSL

## 機能

- 背景動画をウィンドウの形にくりぬいて表示
- ウィンドウはドラッグで自由に移動可能
- マウス座標をリアルタイムで表示
- WebGLシェーダーによるエフェクト（実装中）
